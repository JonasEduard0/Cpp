# ⚙️ C++ - Algoritmos e Estruturas de Dados

Este repositório reúne **anotações, implementações e estudos em C++**, com foco em:

* **Algoritmos**
* **Estruturas de Dados**
* **Análise de Complexidade**
* **Lógica de programação**
* **Implementações clássicas usadas em Ciência da Computação**

O objetivo é manter uma **base organizada de conhecimento em C++**, contendo tanto teoria quanto implementações práticas.

---

# 🎯 Objetivo do Projeto

Este projeto foi criado para:

* Consolidar conhecimentos de **C++**
* Estudar **algoritmos clássicos**
* Implementar **estruturas de dados fundamentais**
* Praticar **análise de complexidade de algoritmos**
* Manter **anotações organizadas**
* Servir como **repositório de estudo e consulta**

---

# 📚 Conteúdo do Repositório

O repositório é dividido em duas partes principais:

## 📄 Anotações
  Um arquivo `.txt` contendo:
* conceitos de algoritmos
* explicações sobre estruturas de dados
* análise de complexidade
* observações sobre C++
* resumos de estudo

# 🧠 Conceitos Estudados

  ## 🔹 Fundamentos de C++
* Sintaxe básica
* Tipos de dados
* Estruturas condicionais
* Estruturas de repetição
* Funções
* Recursão

## 🔹 Estruturas de Dados
  Implementações clássicas como:

* Filas (queue)
* Árvores
* Grafos
* Tabelas de dispersão (hash)

  Exemplos incluídos:

* Binary Search

### Algoritmos de Ordenação

* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

  Estudo de eficiência de algoritmos utilizando:

* Comparação entre algoritmos
